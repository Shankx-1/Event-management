Event Management System (EMS)

Overview

A simple, beginner-friendly Python application to manage events and attendees. This project demonstrates the use of file handling, modular programming, and standard Python libraries.

Features

Create new events (saved to events.csv).

Register attendees (saved to attendees.csv).

View all scheduled events.

View all registered attendees.

Technologies Used

Python 3.x

Library: csv (For storing data like a database)

Library: datetime (For handling event dates)

Library: random (For generating unique IDs)

Library: os (For file checking)

How to Run

Ensure you have Python installed.

Download all .py files into one folder.

Open a terminal/command prompt in that folder.

Run the following command:

python main.py


Follow the on-screen menu instructions.
